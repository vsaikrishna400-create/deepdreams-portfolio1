We encountered 2 build errors, and both are now **FIXED** in your local files. 
You just need to update these 2 files on GitHub to deploy your site successfully.

### 1. Update `Contact/index.tsx`
1. Go to GitHub repo: [vsaikrishna400-create/deepdreams-portfolio1](https://github.com/vsaikrishna400-create/deepdreams-portfolio1)
2. Navigate to: `src` → `components` → `Contact`
3. Click "Add file" → "Upload files"
4. Upload `c:\Portfolio\portfolio-app\src\components\Contact\index.tsx`
5. Commit changes

### 2. Update `Services/index.tsx`
1. Go to: `src` → `components` → `Services`
2. Click "Add file" → "Upload files"
3. Upload `c:\Portfolio\portfolio-app\src\components\Services\index.tsx`
4. Commit changes
