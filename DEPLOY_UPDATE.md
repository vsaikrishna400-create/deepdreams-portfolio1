## 📝 Steps to Update Your Site

You need to update 3 files on GitHub. Vercel will auto-deploy after these changes.

### 1. Update `src/app/layout.tsx`
1. Go to your repo: [vsaikrishna400-create/deepdreams-portfolio1](https://github.com/vsaikrishna400-create/deepdreams-portfolio1)
2. Go to `src` → `app`
3. Click "Add file" → "Upload files"
4. Upload: `c:\Portfolio\portfolio-app\src\app\layout.tsx`
5. Commit changes

### 2. Update `src/components/Hero/index.tsx`
1. Go to `src` → `components` → `Hero`
2. Click "Add file" → "Upload files"
3. Upload: `c:\Portfolio\portfolio-app\src\components\Hero\index.tsx`
4. Commit changes

### 3. Update `src/components/VideoGallery/index.tsx`
1. Go to `src` → `components` → `VideoGallery`
